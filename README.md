 1. ¿Para qué sirve el archivo .gitignore?

Sirve para indicarle a Git qué archivos no debe seguir ni subir al repositorio.

2. ¿Qué tipo de archivos es conveniente ignorar?

Archivos temporales, logs, archivos del sistema, archivos generados automáticamente y datos sensibles.

3. ¿Un archivo ignorado se sube a GitHub?

No, los archivos ignorados no se agregan ni se suben al repositorio.
